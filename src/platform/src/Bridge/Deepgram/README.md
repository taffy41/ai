Deepgram Platform
=================

Deepgram platform bridge for Symfony AI. Supports text-to-speech (TTS) and speech-to-text (STT)
over HTTP.

Deepgram Documentation
----------------------

 * [API overview](https://developers.deepgram.com/reference/deepgram-api-overview)
 * [Text-to-speech (`/v1/speak`)](https://developers.deepgram.com/reference/text-to-speech/speak-request)
 * [Speech-to-text (`/v1/listen`)](https://developers.deepgram.com/reference/speech-to-text/listen-pre-recorded)
 * [Model catalog (`/v1/models`)](https://developers.deepgram.com/reference/manage/models/list)
